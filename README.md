# PlanYourDay
Calendar

## Description

I created this site to help you to manage your day more effectively. You can create events for each hour, excluding the past. 
Each period of time has its own color; past - gray, present - red and future - green.

## Technology stack 

* CSS

* HTML

* JavaScript

* JQuery

* DayJS

* momentjs
